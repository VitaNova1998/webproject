СОЗДАНИЕ САЙТА НА БАЗЕ СИСТЕМЫ УПРАВЛЕНИЯ КОНТЕНТОМ

CMS — программное обеспечение для создания, редактирования, организации структуры и управления WEB-ресурсом.

Особенность работы CMS заключается в возможности добавлять, изменять и редактировать контент сайта без изменения внутреннего механизма организации и вывода WEB-страниц.

Систему управления содержимым сайта можно условно разделить на два хранилища информации: в первом размещены базы данных с контентом WEB-страниц, во втором — элементы визуализации, позволяющие представлять содержимое WEB-ресурса пользователям интернета (шаблоны, графические элементы).

Для удобства работы с контентом большинство систем включает в себя инструменты визуального редактора кода — программы, создающей HTML-код из особой упрощенной разметки, облегчающей форматирование текста.

Основная задача CMS — предоставление возможности самостоятельного наполнения WEB-ресурса информацией без привлечения технических специалистов. Здесь не требуется владения специальными знаниями WEB-технологий и программирования. Для эксплуатации данных систем необходимо освоить принципы работы и ознакомиться с руководством по использованию конкретной CMS.

Преимущества и недостатки использования CMS

Преимущества:

• снижение стоимости технической поддержки;

• минимализация сроков разработки;

• легкая настройка параметров управления контентом;

• простота использования программ.

Недостатки:

• большой объем информации, который может вызывать перегрузку сервера;

• нестабильность генерируемых URL страниц, что приводит к ухудшению репутации у поисковых систем;

• временные затраты на освоение особенностей работы CMS;

• расходы на поддержку работы CMS (часто приходится устанавливать дополнительные модули и шаблоны, которые могут оказаться платными).
