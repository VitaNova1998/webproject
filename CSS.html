Связанные стили

При использовании связанных стилей описание селекторов и их значений располагается в отдельном файле с расширением .css, а для связывания документа с этим файлом применяется тег link. Данный тег помещается в контейнер head.

Значение атрибута тега link — rel остаётся постоянным. Значение href указывает путь к CSSфайлу, он может быть задан как относительно, так и абсолютно. Таким образом, существует возможность подключить таблицу стилей, которая находится на другом домене. Содержимое файла main.css, подключенного с помощью тега link, приведено на риcунке

 

Файл со стилем не хранит никаких данных, кроме синтаксиса CSS, а HTML-документ содержит только ссылку на файл со стилем. Таким способом реализован принцип разделения кода и оформления сайта. Использование связанных стилей является наиболее универсальным и удобным методом добавления стиля на сайт. Ведь стили хранятся в одном файле, а в HTML-документах указывается только ссылка на него.

Глобальные стили

При использовании глобальных стилей свойства CSS описываются в самом документе и располагаются в заголовке WEB-страницы. По гибкости и возможностям этот способ добавления стиля уступает связанному, но этот способ также позволяет хранить стили в одном месте, в данном случае прямо на той же странице с помощью контейнера style

На практике данный способ используют очень редко и только в исключительных случаях.

Внутренние стили

Внутренний стиль является расширением для одиночного тега, используемого на текущей WEBстранице. Для определения стиля используется атрибут style, а его значением выступает набор стилевых правил

Внутренние стили рекомендовано применять ограниченно или лучше вообще отказаться от их использования. Использование таких стилей увеличивает общий объём файлов, что, в свою очередь, ведет к увеличению времени загрузки страницы в браузере, а также усложняет внесение изменений.

Все описанные методы использования CSS могут быть применены в сочетании друг с другом, но в этом случае необходимо помнить об их иерархии. Первым имеет приоритет внутренний стиль, затем глобальный стиль и в последнюю очередь связанный стиль.

Импорт CSS

vЕсли у вас уже есть готовый CSS-файл с описанными стилями, то, чтобы не копировать код, можно импортировать содержимое CSS-файла с помощью команды @import.

Этот метод допускается использовать совместно со связанными или глобальными стилями, но нельзя использовать с внутренними стилями.

Пример синтаксиса:

@import url(«имя файла»);

@import «имя файла»;

После ключевого слова @import указывается путь к стилевому файлу одним из двух приведенных способов — с помощью url или без него. На рис. 7 и 8 показано, как можно импортировать стили.
